# Introduction {#mainpage}

**JPL Spatial** is an engine-independent C++ library for audio spatialization, sound propagation, and related DSP. It does not own the audio device or mixing backend. Applications can use their own vector and scene-query types.

The core library **has no external dependencies**.

## Getting started

To get started, take a look at the [HelloJPLSpatial](https://github.com/Jaytheway/JPLSpatial/tree/main/Examples/HelloJPLSpatial) example.
It shows how to integrate JPL Spatial in a CMake project and use JPL Spatial [VBAP Panner](@ref JPL::VBAPannerBase).

For instructions on how to integrate various layers of JPL Spatial API, see [Integrating JPL Spatial](integration.md).
